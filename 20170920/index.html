# 動手做
## 在 XAMPP Controll Panel ## 開啟 Explorer
## 進到 htdocs ## 刪光裡面的檔案
## 在網址列輸入 `cmd` ### 在 terminal 輸入 `code .` (VSCode only)
## 新增一個 index.html ## 輸入一個基本的 HTML 
<!doctype html>
<html>
    <head>
        <meta charset="utf-8">
    </head>
    <body>
        <h1>
            我也搞不定自己
            那矛盾的個性
            我想來想去結論就是喜歡討厭你

    </body>
</html>
