## 命名東西  
  
永遠是最難的部分  
  
---  
  
![bl](./black-quation.jpg)  

---  

``` java  

// bad  
String a = "Hello";  
for (i = 0; i < 3; i++ ){  
  for (j = 0; j < 3; j++){  
    // do something  
  }  
}  
  
// good  
String greet = "Hello";  
for (row = 0; row < 3; row++ ){  
  for (column = 0; column < 3; column++){
        // do something  
  }  
}  

```
